Help

Welcome to the JS Live Workbook. The porpose of this app is to make it easy to try out code and do experiemnts while you are reading a tutorial or doing coding exercises. The app combines:

  • A file viewer for HTML and Markdown documnts
  • A code editor
  • A JavaScript console for running code
  • A command line for imediate execution of code in the console

Here are common actions:

File Viewer Menu

  • Open Markdown: Load and preview a local .md file in the left pane with syntax highlighting.
  • Open HTML: Load and preview a local .html file in the left pane.
  • Open from URL: Load HTML or Markdown content directly from a URL (local or remote).
  • Recent Files: Quick access to recently opened files (appears in dropdown when available).

Editor Menu

  • Open File: Open a local .js or .txt file into a new editor tab.
  • Save File: Save the current editor tab contents (supports File System Access API where available, with download fallback).
  • Hide/Show Editor: Toggle visibility of the editor panel to maximize the file viewer area.

Editor Features

  • Tabbed Editing: Work with multiple files simultaneously using tabs. Click the + button to create a new tab.
  • Run Code: Executes the current editor code in a persistent sandbox so variables are available in the REPL.
  • Clear Editor: Clears all code from the current editor tab.
  • Clear Console: Clears the console output.

Console & REPL

  • Console Output: Displays results from running code, including console.log() output.

Layout

  • Resizable Panels: Drag the vertical divider between the file viewer and editor to adjust column widths. Your preference is remembered between visits.
